## ✨ Core Features

| 🛠 Tool | Description |
| :--- | :--- |
| **Integrated Search** | Universal search across Google, DuckDuckGo, Bing, Brave, YouTube, Wikipedia, and more. Features built-in voice typing. |
| **Personalized Themes** | Hand-pick your style from premium themes or use the custom color picker for dynamic UI tinting. |
| **Dynamic Wallpapers** | Upload your own or enjoy daily random photography powered by [Lorem Picsum](https://picsum.photos). |
| **Live Widgets** | Gorgeous analog & digital clocks, alongside real-time weather tracking (°C / °F). |
| **AI Integration** | One-click access to ChatGPT, Gemini, Copilot, Perplexity, Claude, DeepSeek, and more. |
| **Productivity Hub** | Features a priority-aware **To-Do List** with daily resets, plus quick shortcuts to your favorite apps. |
| **Seamless Google Apps** | Launch Gmail, Drive, Docs, Calendar, and more right from your homepage. |

## 🚀 Installation

We support all major Chromium and Firefox-based browsers.

### 🛠 Manual / Developer Installation

Want the cutting-edge version straight from the source?

1. Clone this repository:
   ```bash
   git clone https://github.com/M-ayank2005/NewHome.git
   ```
2. Open your browser's extension manager:
   - Chrome/Brave: `chrome://extensions`
   - Edge: `edge://extensions`
3. Enable **Developer Mode**.
4. Click **Load Unpacked** and select the cloned folder.
